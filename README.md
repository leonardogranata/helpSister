## 💜 O que é a HelpSister?

A **HelpSister** é uma plataforma web que conecta **babás** a **contratantes** (pais e responsáveis) de forma segura, transparente e baseada em reputação real.

Aproveitar momentos com a família deveria ser simples. Mas, na prática, encontrar uma babá disponível, preparada e verdadeiramente comprometida ainda é um processo precário — cheio de indicações informais, avaliações superficiais e nenhuma garantia real. O que deveria ser uma busca simples vira uma fonte de preocupação e insegurança.

A HelpSister nasce para resolver exatamente isso.

---

## 🔍 O Problema

Hoje, contratar uma babá envolve:

- **Falta de informações confiáveis** sobre a profissional
- **Avaliações superficiais ou inexistentes** — ou simplesmente nenhuma
- **Dificuldade de avaliar consistência** — uma boa avaliação antiga não diz nada sobre agora
- **Risco alto para ambas as partes**, especialmente em relação à segurança

O mercado é informal. A transparência é quase zero.

---

## ✅ A Solução

A HelpSister resolve isso com um sistema que vai além de "conectar pessoas":

- **Perfis detalhados** de babás e contratantes, construídos a partir de questionários estruturados
- **Sistema de avaliação contínua e transparente**, baseado em histórico real de serviços
- **Métricas de confiabilidade dinâmicas** — não só uma nota estática, mas um indicador que reflete o comportamento recente
- **Regras e boas práticas** visíveis e integradas à plataforma, promovendo profissionalismo de ambos os lados
- **Histórico de atuação** consultável por qualquer contratante

## 👥 Para quem é?

**Contratantes (pais e responsáveis)**
Famílias que precisam contratar babás com segurança, seja para o dia a dia, eventos, emergências ou ocasiões especiais — e que querem ter certeza de que seus filhos estão em boas mãos.

**Babás (candidatas)**
Profissionais que querem encontrar trabalho com mais credibilidade, segurança e visibilidade — e construir uma reputação real baseada em desempenho.

---

## 🧩 Funcionalidades (MVP)

### Para candidatas
- Cadastro e autenticação
- Perfil completo com apresentação pessoal, experiências, postura profissional e disponibilidade
- Questionário estruturado de onboarding (experiência, situações reais, capacitação, estilo)
- Histórico de serviços realizados
- Rendimento por dia trabalhado
- Escala de trabalho (quais dias irá trabalhar)
- Chat com contratantes
- Acesso às regras e boas práticas da plataforma

### Para contratantes
- Cadastro e autenticação
- Perfil da família com informações das crianças e rotina
- Questionário estruturado (expectativas, estilo familiar, preferências com a babá)
- Visualização de perfis de candidatas
- Sistema de avaliação após os serviços
- Chat com candidatas
- Indicador de confiabilidade das profissionais

### Regras e boas práticas (módulo dedicado)
A plataforma conta com uma seção exclusiva de conduta profissional, visível para ambas as partes, cobrindo pontualidade, privacidade, segurança, postura, higiene, conduta ética e muito mais.

## ⚙️ Como instalar e rodar

### Pré-requisitos

- Python 3.12+
- Node.js 20+
- npm

### 1. Clonar o repositório

```bash
git clone https://github.com/leonardogranata/helpSister.git
cd helpSister
```

### 2. Backend (Django)

```powershell
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

O backend estará disponível em:
- API: `http://127.0.0.1:8000`
- Admin: `http://127.0.0.1:8000/admin/`

### 3. Frontend (React)

Em outro terminal:

```powershell
cd frontend
npm install
npm run dev
```

> Se o PowerShell bloquear o `npm.ps1`, use: `cmd /c npm run dev`

O frontend estará disponível em `http://127.0.0.1:5173`

## 📌 Observações técnicas

- O frontend salva sessão em `localStorage` com as chaves `hs_token` e `hs_user`
- Upload de imagem de perfil usa `MEDIA_URL=/media/` e `MEDIA_ROOT=backend/media/`
- O proxy do Vite redireciona `/api/*` e `/media/*` para `http://127.0.0.1:8000`
- Páginas HTML legadas do Django foram removidas — o fluxo atual é 100% React + API REST
